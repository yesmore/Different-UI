# 介绍




Different UI - A Vue.js 3 UI library made by @yesmore

  • 🌈 With Colorful Style
  • 💪 Vue 3 Composition API
  • 🔥 Written in TypeScript

这个项目仍在持续开发中。欢迎加入开发，并提出您的第一次 Pull Request 。
